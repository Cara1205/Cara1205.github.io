更换hexo主题

– – – – – – – – – – – – –2018.11.10



序言

以后打算多在hexo上记录一些自己的进步史（以做复习和回顾）

(突然发现html5上的语言MP上也能用呢ww，第一次写博文时不知道怎么做到的换行用br做到了）

此文内容有参考很多其他的博文

Step 1：安装主题

在网站上找到自己中意的Hexo主题，这里贴一个比较全的网站：

知乎话题：有哪些好看的 Hexo 主题？

注：点进右方的那个里面有许多文件的网址

我个人使用的是pacman主题，以此为例



在\hexo的本地目录中用Git bash,输入以下命令

$ git clone https://github.com/A-limon/pacman.git themes/pacman

或者到\themes目录下执行

$ git clone https://github.com/A-limon/pacman.git


Step 2:修改配置文件，安装

修改目录\hexo_config.yml中的theme属性，将其设置为pacman（冒号后的空格不可少）

theme: pacman


Step 3：更新主题

关于这一步我有一点不确定，在我看到第一个博客中称，只需在themes\pacman目录下执行以下命令

$ git pull origin master

就完成，没有后续的生成静态页面、上传至git等工序，可能是该博客主自行完成了其他步骤，没有说明罢。

而在另一博客中， 则没有上一步的git pull origin master,而是直接在本地查看无误后，输入生成发发布命令，就可将新主题发布到博客网站上了。



作为复习，回顾一下关于hexo的命令吧

$ hexo g    #等同于hexo generate,生成静态文件
$ hexo s    #等同于hexo server，在本地服务器运行
$ hexo d    #等同于hexo deploy，上传至git(?)

  • 最好在生成静态文件前，执行清除缓存的命令，

    $ hexo clean

完成

做完这些之后，你就可以打开你的博客网站，查看新换的主题啦。刚刚上传完时可能网页会有一些异常，稍等一下就好。
